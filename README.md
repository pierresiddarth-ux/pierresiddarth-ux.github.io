# Embedded Video Stream Website

A clean, responsive single-page website that embeds a video player.

## How to Host on GitHub Pages

1. Create a repository on GitHub named `<your-username>.github.io`.
2. Push this project to your repository:
   ```bash
   git add index.html
   git commit -m "Add responsive embedded stream"
   git branch -M main
   git remote add origin https://github.com/<your-username>/<your-username>.github.io.git
   git push -u origin main
   ```
3. Go to **Settings** > **Pages** in your GitHub repository and ensure the source is set to build from the `main` branch.
4. Your website will be live at `https://<your-username>.github.io`.
